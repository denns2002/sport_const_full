Translate admin panel¶

translate_ru(eng='', ru='')¶

Returns the English or Russian translation, depending on the LANGUAGE_CODE setting.

If only one of the variables is specified, then returns it. If the eng and ru parameters are not specified, returns the boolean value of the settings LANGUAGE_CODE == «RU».

Параметры:

  • eng (str or list or tuple) – English word(s)

  • ru (str or list or tuple) – Russian translation

Результат:

translation or bool LANGUAGE_CODE setting.

Тип результата:

bool or str or list or tuple

Исключение:

  • ValueError – if length eng, ru does not match

  • TypeError – if eng, ru is not of str, list, tuple types

def translate_ru(eng="", ru=""):
    types = (str, list, tuple)

    if not (isinstance(eng, types) and isinstance(ru, types)):
        raise TypeError("Unsupported type!")

    if eng or ru:
        return ru if not eng or settings.LANGUAGE_CODE == "RU" and ru else eng

    else:
        return settings.LANGUAGE_CODE == "RU"
